# Zibyan Satopay — Portfolio

A professional portfolio for Muhammad Zibyan Feroz Satopay, focused on mechatronics, IoT, embedded control, automation, and connected product engineering.

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```

Built with React, Vite, Framer Motion, and React Icons.
